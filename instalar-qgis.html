Como instalar o QGIS

Instruções passo a passo

O QGIS é um software livre e gratuito, disponível para Windows, MacOS X e Linux. Veja como instalar abaixo:

Windows:

  1. Baixe o Instalador Standalone QGIS Versão 2.14 (32 bits) ou Instalador Standalone QGIS Versão 2.14 (64 bits).
    • Não sabe se seu computador é 32 bits ou 64 bits? Siga estas instruções: https://support.microsoft.com/pt-br/kb/827218
  2. Abra o instalador que você acabou de baixar. Clique em próximo, concorde com os termos da licença.
  3. Escolha a pasta de destino e clique em próximo.
  4. Selecione os componentes a instalar (eles são opcionais e não necessários para o curso).
  5. Clique e instalar e aguarde.
  6. Pronto! Para abrir o QGIS digite "QGIS" na barra iniciar e abra o "QGIS Desktop 2.14"

Mac:

  1. Baixe este arquivo: http://www.kyngchaos.com/files/software/qgis/QGIS-2.14.8-1.dmg
  2. Abra o arquivo que você acabou de baixar e você deve ver uma pasta assim:
  3. Instale cada um dos arquivos .pkg na ordem numérica indicada no nome dos arquivos.
    • 1 GDAL Complete.pkg
    • 2 NumPy.pkg
    • 3 matplotlib.pkg
    • 4 Install QGIS.pkg
  4. Pronto!
