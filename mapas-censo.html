Mapeando com dados do Censo

Curso prático intensivo

Descrição

O Censo, produzido a cada 10 anos pelo IBGE, é uma rica fonte de dados sobre os municípios brasileiros, embora não esteja em um formato exatamente amigável. Este curso prático oferecido pelo laboratório Medida SP vai te ensinar a compreender melhor o Censo e trabalhar com seus dados brutos para criar mapas. Para isso vamos utilizar o software livre e gratuito QGIS.

O que você vai aprender no curso?

  • Baixar dados do Censo;
  • Visualizar e mapear dados do Censo;
  • Trabalhar com tabelas no QGIS;
  • Fazer junções de tabelas;
  • Fazer cruzamentos espaciais de dados;
  • Cruzar dados do Censo com outras bases de dados;
  • Organização de dados.

Para quem é o curso?

O curso se destina a todos que se interessam por mapeamento e visualização de dados, sobretudo para questões urbanas, em especial àqueles que têm interesse por explorar dados do Censo.

Que tipos de dados fazem parte do Censo?

O Censo, produzido pelo IBGE a cada dez anos, possui uma série de informações sobre a população e domicílios, incluindo, entre outros:

  • Moradores, com dados sobre raça, idade e gênero;
  • Tipo dos domicílios;
  • Renda;
  • Abastecimento, incluindo esgoto, lixo, água e energia;
  • Infraestrutura urbana, como pavimentação, iluminação, arborização e calçadas.

Pré-requisitos

O curso pressupõe uma experiência prévia de nível básico com QGIS ou outro software de geoprocessamento (por exemplo, ArcGIS). Se você não tem nenhuma experiência, recomendamos que faça primeiro nosso curso de Introdução a GIS.

Como funciona o curso?

Começamos com uma exposição rápida dos conceitos básicos de dados, mapeamento e GIS. A partir daí, realizamos uma sequência de exercícios práticos que aprofundam e introduzem novos conceitos. Ao fim da aula os alunos terão como produtos mapas e análises que refletem todo conteúdo aprendido.

Que software vamos utilizar?

QGIS (Quantum GIS), um software livre e gratuito. O QGIS suporta formatos vetoriais, raster, banco de dados, entre outros. Com ele você poderá visualizar, editar, analisar, exportar e gerenciar dados. Além disso, o projeto QGIS possui um ecossistema vibrante de contribuidores que melhoram o software continuamente e desenvolvem plugins e novas ferramentas.

O que você precisa trazer?

Somente seu computador com o QGIS instalado (é gratuito e vamos te ensinar a instalá-lo).
